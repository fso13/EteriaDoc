# Руководство по стилю форматирования Этерии

Единый стиль для всех текстов кампании. Эталон — «Хозяин леса».

## Метки и заголовки

### Категории (локации, типы контента)
**Тройное выделение** для меток-категорий:

```
***В лесу***: Герои находят...
***От НПС***: Старый трактирщик рассказывает...
***Окружение***: Квартал живёт размеренной жизнью...
```

### Игровые механики
**Двойное выделение** для игровых терминов:

```
**Цель**: Найти мастера Элрика
**Задача**: Проникнуть в часовню
**Босс**: Хозяин Леса
**Награда**: Арбалет и Нож
**Столкновение**: Группа натыкается на...
```

### Подзаголовки
*Курсив* для подзаголовков внутри блока: `*Диалог*:`, `*Внешность*:`

### Нумерованные структурные параграфы
Этапы квеста, сцены, разделы — как заголовки `###`:

```
### 1. Подход к Часовне (Создание атмосферы)
### 2. На подступах (Небоевое взаимодействие)
### 3. Внутри Часовни (Исследование и атмосфера)
```

В исходнике: `1\. Title` — скрипт форматирования преобразует в `### 1. Title`.

## Цитаты и диалоги

Реплики в `<note>`, прямая речь в *курсиве* внутри. Ремарка: `НПС (действие):`

## Списки

- Один пробел после дефиса: `- `
- **Метка:** в начале при необходимости

## Описания

- *Курсив* — атмосфера, мысли
- **Жирный** — имена, термины
- «Ёлочки» — названия, цитаты
